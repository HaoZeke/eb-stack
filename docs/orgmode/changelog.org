The project has no public release. The complete unreleased change list lives
in [[https://github.com/HaoZeke/eb-stack/blob/main/CHANGELOG.md][CHANGELOG.md]].

The version-one design provides:

- one canonical package plan for conda-forge, Spack, and EasyBuild;
- planned CycloneDX plus per-profile Resolvo locks;
- one conventional EasyBuild recipe per installable profile;
- in-solver stack pin fallback and candidate exclusions;
- layered target routing and persisted typed build campaigns;
- independent resolves, builds, and binary-verified claims;
- matching CLI, MCP, new-package skill, and annual-bump skill workflows.

The unreleased scaffold/compatibility command surface is removed rather than
maintained alongside the canonical workflow.
